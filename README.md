# ExchangeServer-
>.ExchangeServer   版本域内网IP信息探测\
>.自行打包\
>.goreleaser build --snapshot 手动编译

>.用法: main.exe -url <IP>



>.选项:\
>.  -url string   指定要处理的IP地址\
>.  -v            显示当前应用程序的版本\
>.  -f            获取给定URL或IP的完全限定域名（FQDN）\
>.  -ip           获取给定URL或IP的内网IP地址\
>.  -p            指定是否使用代理\
>.  -h            显示此帮助信息
